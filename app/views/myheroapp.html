MyHero Application Architecture

What is MyHero?

MyHero is a very simple sample application developed using a "Cloud Native" design method to be used for demonstration and education purposes. It is composed of several microservices, each leveraging standard API interaction. The simplest deployment of MyHero consists of just three services, and when deployed with all features, it can be as many as ten different services.
